# MovieDb Web Application

This project is a React.js web application for browsing movies, as per the requirements outlined in the provided document.

## Features
Home Page / Popular Movie Page:-Displays a list of popular movies with their posters, titles, and ratings. 
Top Rated Movie Page:- Shows a list of top-rated movies. 
Upcoming Movie Page:- Presents a list of upcoming movies. [cite: 3]
Single Movie Detail Page:- Provides detailed information about a selected movie, including:
     Title and rating 
     Runtime and genres 
     Release date 
    Cast information 
Searched Movie Page:-Displays search results for movies based on user input 
Global Search:- A search bar in the Navbar allows users to search for movies from any page. 
Pagination:- The application includes pagination to navigate through large lists of movies
Responsive Design:-The application is designed to be responsive and work well on various screen sizes. 

## Technologies Used

 React.js
 React Router DOM (for navigation)
 CSS (for styling)
 The Movie Database (TMDb) API (for movie data)

## Setup

1.  Clone the repository.
2.  Install dependencies: `npm install
3.  Start the development server: `npm run dev'
4.  Open the application in your browser.
